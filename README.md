# Tech Solutions Landing Page

## Description
A professional and responsive landing page for a technology solutions company. This website serves as a digital storefront for the company, showcasing their services, expertise, and providing potential clients with an easy way to get in touch. The site features a clean, modern design with a focus on user experience and mobile responsiveness.

## Features
- **Home Section**: Engaging hero section with call-to-action buttons
- **About Us**: Company overview and mission statement
- **Services**: Technology solutions including:
  - Computer Repair
  - Website Development
  - Robot Manufacturing
  - Software Development
- **Contact Form**: Easy-to-use contact form for inquiries
- **Responsive Design**: Fully optimized for all device sizes (mobile, tablet, desktop)

## Technologies Used
- HTML5
- CSS3 (with Flexbox and Grid layouts)
- Google Fonts
- Font Awesome icons
- CSS Animations

## Installation and Usage
1. **Clone the repository**
   ```
   git clone https://github.com/Basant1Saini/Landing-page.git
   ```

2. **Navigate to the project directory**
   ```
   cd Landing-page
   ```

3. **Open in browser**
   - Simply open the `index.html` file in your preferred web browser

4. **Development**
   - Edit HTML files to modify content
   - Update CSS files in the `styles.css` file to change styling
